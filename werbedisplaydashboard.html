Werbedisplay - Dashboard

Das Abschlussprojekt meiner Ausbildung

Die Erstellung eines Dashboardes zur Steuerung der Inhalte auf dem Werbedisplay

Problem

Für die Inhalte auf dem Werbedisplay ist die Marketingabteilung zuständig. Wenn dem Display neue Inhalte hinzugefügt, oder bestehende Inhalte geändert werden sollen, muss sich das Marketing an die IT-Abteilung wenden. Um die Inhalte zu ändern müssen Dateien auf einen Server gelegt und ggf. Quellcode vom Anzeigesystem angepasst werden.

Ziel / Lösung

Mit dem Dashboard kann die Marketingabteilung eigenständig, ohne Umweg über die IT, die Inhalte auf den Werbedisplays anpassen. Es können Texte und Bilder hinzugefügt und wieder entfernt werden. Damit nur bestimmte Personen Zugriff auf das Dashboard haben, wird beim Aufruf der Seite nach einem Benutzernamen und Passwort gefragt. Nur wenn man die Zugangsdaten korrekt eingibt, kommt man auf das Dashboard. Dies wird mit einer Access-List im Nginx Proxy Manager realisiert.

Programmiersprachen / Techniken
