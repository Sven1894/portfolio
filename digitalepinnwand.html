Digitale Pinnwand

Erstellung einer digitalen Pinnwand mit dazugehörigen Dashboard

Problem

Eine digitale Pinnwand für die IT-Abteilung soll erstellt werden, damit jeder einen Überblick über die aktuellen Aufgaben und anstehende Termine hat.

Ziel / Lösung

Die digitale Pinnwand beruht, wie auch das Anzeigensystem für die Werbedisplays, auf dem Open-Source-Projekt MagicMirror2. Mit dem Modul "MMM-Memo" habe ich der digitalen Pinnwand für jeden Kollegen und jede Kollegin je einen Notizzettel hinzugefügt. Darauf können Aufgaben für die jeweilige Person eingetragen werden. Die Inhalte der Notizzettel können per HTTP-Get-Anfragen verwaltet werden. Zudem habe ich noch einen Newsfeed und den Bereichskalender, so wie Wetter und Uhrzeit eingebunden. Die Pinnwand wird auf einem Display im Büro angezeigt, kann aber auch von jeden Kollegen und jeder Kollegin auf dem Computer aufgerufen werden. Um das Verwalten der Notizzettel zu vereinfachen, habe ich mit HTML, CSS und JavaScript ein Dashboard erstellt, mit dem man einfacher Notizen hinzufügen, entfernen und als wichtig markieren kann.

Programmiersprachen / Techniken

HTML, CSS, JavaScript
