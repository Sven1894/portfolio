Werbedisplays

Erstellung eines Anzeigesystems für Werbedisplays

Problem

Ein Anzeigesystem für Werbedisplays soll erstellt werden. Darauf sollen Werbung in Form von Grafiken, Schlagzeilen aus einem RSS-Feed und Texte dargestellt werden.

Ziel / Lösung

Das Anzeigesystem beruht auf dem Open-Source-Projekt MagicMirror2. Damit kann man Smarte Spiegel bauen, die einem wichtige Informationen, wie Uhrzeit, Wetter, Verkehrslage, uvm. im Spiegel anzeigen. Es lässt sich aber durch einige Anpassungen auch zu einem solchen Anzeigesystem umfunktionieren. Bei der Grundinstallation werden schon einige Standardmodule, wie z.B. Uhrzeit oder Wetter, mitgeliefert. Durch die große und aktive Community gibt es für alles Mögliche ein Modul: Für den Verkehr, Sportergebnisse, Nachrichten, Bahnverbindungen uvm. Diese Module kann jeder dann nochmal genau auf die eigenen Bedürfnisse anpassen.
Für das Anzeigensystem habe ich jeweils ein Modul für die Uhrzeit, das Wetter, aktuelle Schlagzeilen, Texte und Bilder benutzt. Diese habe ich so angepasst, dass sie genau das erfüllen, was sie in meinem Anwendungsfall sollen.

Programmiersprachen / Techniken

JavaScript, CSS, HTML
